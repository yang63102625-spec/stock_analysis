# 选股策略优化完成 — 实时行情筛选

## 📋 优化总结

你的选股系统现已支持**实时行情筛选**，解决了"只能用日线数据，不支持当天操作"的问题。

### 核心改进

**之前的流程**：
```
日线数据 → 量化筛选（60天趋势/均线等）→ 候选池 → AI精选 → 推送
                    ⚠️ 只能在收盘后运行
```

**优化后的流程**：
```
日线数据 → 量化筛选 → 【新增】实时筛选 → AI精选 → 推送
                     ✓ 当天下午可用
                     ✓ 支持当天买、第二天卖
```

## 🎯 主要功能

| 功能 | 说明 |
|------|------|
| 排除涨停 | 自动过滤已涨停的股票（>9.5% or >20%） |
| 排除跌停 | 自动过滤已跌停的股票（<-9.5% or -20%） |
| 涨幅限制 | 可配置当日涨幅范围（如 -2% ~ 8%） |
| 放量过滤 | 可配置最大量比（如 >5 的排除） |
| 灵活配置 | 所有规则都在 `.env` 中配置，无需改代码 |

## ⚡ 快速启用

### 方式一：一行启用（推荐）

在 `.env` 中添加：
```bash
PICKER_ENABLE_REALTIME_FILTER=true
```

就这一行！系统会用默认规则：排除涨停/跌停，不限制涨幅、不过滤放量。

### 方式二：精细化配置

```bash
# 开启实时筛选
PICKER_ENABLE_REALTIME_FILTER=true

# 排除涨停/跌停
PICKER_REALTIME_EXCLUDE_LIMIT_UP=true
PICKER_REALTIME_EXCLUDE_LIMIT_DOWN=true

# 只要当日涨 -2% ~ 8% 的股票（避免追太高）
PICKER_REALTIME_DAILY_CHG_MIN=-2
PICKER_REALTIME_DAILY_CHG_MAX=8

# 排除异常放量（量比 > 5）
PICKER_REALTIME_MAX_VOLUME_RATIO=5.0
```

## 📝 配置说明

| 环境变量 | 类型 | 默认 | 说明 |
|---------|------|------|------|
| `PICKER_ENABLE_REALTIME_FILTER` | bool | true | 总开关 |
| `PICKER_REALTIME_EXCLUDE_LIMIT_UP` | bool | true | 排除涨停 |
| `PICKER_REALTIME_EXCLUDE_LIMIT_DOWN` | bool | true | 排除跌停 |
| `PICKER_REALTIME_DAILY_CHG_MIN` | float | 空 | 最小涨幅(%)，空=无下限 |
| `PICKER_REALTIME_DAILY_CHG_MAX` | float | 空 | 最大涨幅(%)，空=无上限 |
| `PICKER_REALTIME_MAX_VOLUME_RATIO` | float | 0 | 最大量比，0=无限制 |

## 💡 使用场景

### 场景 1：每天下午 3 点自动选股（最实用）

GitHub Actions 定时 15:00 运行选股：

```bash
# .env
PICKER_ENABLE_REALTIME_FILTER=true
PICKER_REALTIME_DAILY_CHG_MIN=-3
PICKER_REALTIME_DAILY_CHG_MAX=8
PICKER_REALTIME_MAX_VOLUME_RATIO=5.0
```

**效果**：选出当天温和上涨、量能适中的股票，推送给你，可以立即下单。

### 场景 2：收盘后补充选股

```bash
PICKER_ENABLE_REALTIME_FILTER=true
# 排除涨停即可，不限制涨幅
```

### 场景 3：追踪强势板块（激进型）

```bash
PICKER_ENABLE_REALTIME_FILTER=true
PICKER_REALTIME_DAILY_CHG_MIN=5         # 只要涨 5% 以上的
PICKER_REALTIME_MAX_VOLUME_RATIO=10.0   # 接受大放量
```

- Tencent（腾讯财经）
